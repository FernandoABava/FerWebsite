Visuales

Este espacio fue creado para poder mostrar y compartir algunas de las animaciones que he creado con el lenguaje de shaders GLSL. Lo aprendí con la intención de poder utilizarlo para generar visuales en tiempo real, por lo que la mayoría de lo que se puede ver acá fue creado a partir de un mismo archivo que fue evolucionando y complejizándose.
Cada una de las visuales que aquí se expone es parte de un largo devenir, durante el cual programo, no pensando en un resultado final, sino jugando con los cambios que se generan en cada momento. De esta forma, cada animación individual no es más que una foto de un momento especcífico del proceso. De la misma forma, hay muchas visuales que han quedado plasmadas en forma de imágenes o videos.
Abajo a la derecha encontrarán un botón para poder ver únicamente la visual.

Ver más

El archivo que utilizo para programar las visuales contiene funciones para gener patrones (Random, Noise, FBM, Voronoi), algunas para generar formas o polígonos, y también funciones para combinar dos fuentes, basadas en la función "blendMode" de processing. Todas estás funciones las trabajo afuera de la función principal, para que el trabajo se realice una vez por cada canal de color. Esto me permite jugar con la fase de cada color y generar combinaciones extrañas.
Además, he construído otro archivo para que modifique imágenes (lo que implica cambiar el output de algunas funciones de float a vec3), como así también armé un programa de processing para controlar la fase de cada color y el tiempo, desde un arduino.
